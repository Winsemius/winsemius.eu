Performance-based Modes

Replacing BVLOS/VLOS/HLOS with performance-based operational modes.

We No Longer Call It BVLOS​

BVLOS/VLOS/HLOS are terms from manned aviation that don't fit autonomous systems. They focus on human visibility rather than system capabilities.

Four Performance Modes​

1. SLOS-DAA (Sensor Line of Sight + Detect & Avoid)​

Concept: System has sensor coverage (not human eyes) + DAA capability.

Sensors:

  • EO/IR cameras (10+ km range)
  • Radar (5-20 km)
  • Acoustic (rotor sound detection)

DAA (Detect & Avoid):

  • Cooperative (ADS-B, transponders)
  • Non-cooperative (radar, visual AI)
  • Automatic collision avoidance

Use cases:

  • ISR missions (persistent surveillance)
  • Maritime patrol (USV, ASW)
  • Border monitoring

2. ELOS-COOP (Electronic Line of Sight, Cooperative)​

Concept: Datalink maintained + cooperative awareness with other aircraft.

Datalink:

  • RF mesh (tactical)
  • LEO satcom (beyond-horizon)
  • 5G/LTE (infrastructure-based)

Cooperative:

  • ADS-B transponder
  • Mode S
  • Mesh networking (swarm coordination)

Use cases:

  • Controlled airspace operations
  • Swarm coordination
  • Joint ops (manned + unmanned)

3. GEO-ASSURED (Geographically/Temporally Assured)​

Concept: Geographic and/or temporal separation ensures safety.

Geographic:

  • Restricted airspace (military ranges)
  • Segregated corridors (drone highways)
  • Altitude blocks (below/above manned traffic)

Temporal:

  • Time-based separation (NOTAMs)
  • Scheduled operation windows
  • Dynamic airspace allocation

Use cases:

  • Training ranges
  • Test & evaluation
  • Offshore operations (oil/gas, wind)

4. PROC-SEG (Procedurally Segregated)​

Concept: Procedural separation like IFR, via ATC coordination.

Procedures:

  • Flight plans filed (like IFR)
  • ATC clearances
  • Separation standards (3-5 NM horizontal, 1000 ft vertical)

Integration:

  • UTM (UAS Traffic Management)
  • Existing ATC systems
  • Deconfliction with manned traffic

Use cases:

  • Long-range logistics (cargo drones)
  • Medical transport (SAR, ambulance)
  • Infrastructure inspection (power lines, pipelines)

Comparison: Old vs New​

AspectBVLOS/VLOS (Old)Performance Modes (New)
FocusHuman visual capabilitySystem sensor capability
FramingPilots' perspectiveTech performance
Safety basis"Pilot can see"Sensor + DAA + procedures
RegulationRestrictive (preserve pilot monopoly)Performance-based
InnovationSlow (regulatory lag)Fast (tech-driven standards)

Military Relevance​

For military ops, BVLOS/VLOS are irrelevant:

  • Combat ranges exceed visual limits
  • All ops are "beyond visual" by definition
  • Focus on sensor performance + cooperative awareness

Performance modes are directly applicable:

  • SLOS-DAA: ISR, strike missions
  • ELOS-COOP: Swarm ops, joint missions
  • GEO-ASSURED: Training, test ranges
  • PROC-SEG: Strategic transport, logistics
